# tools
Collection of useful scripts


### `git_push.sh`
Pushes any changes in this directory to git.
If you want to have it automatically updated every hour by this script add this line 
to your crontab (by running `crontab -e`)
```
* * * * * /<path>/<to>/git_push.sh
```
